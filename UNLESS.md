## The Conjunction of Exception

Unless: except if. Only if not.

---

## The Logic of Unless

"A unless B" means:
A will happen, except if B happens.
If B, then not-A.
If not-B, then A.

The unless is a conditional in reverse.
Instead of "if this, then that"—
"that, unless this."

The direction matters.
The emphasis differs.
Unless puts the exception first in mind.
